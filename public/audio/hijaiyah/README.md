# Audio Files for Hijaiyah

Place MP3 audio files for each hijaiyah letter here.

## File Naming Convention:
- `alif.mp3`
- `ba.mp3`
- `ta.mp3`
- ... etc

## With Harakat:
- `alif-fathah.mp3` (اَ)
- `alif-kasrah.mp3` (اِ)
- `alif-dhammah.mp3` (اُ)
- ... etc

## Sources for Arabic Audio:
- Record your own
- Use royalty-free Islamic education resources
- Use TTS services like Google Cloud TTS (Arabic voice)
